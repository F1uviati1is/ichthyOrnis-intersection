## Name short
Fluvial Network X Water Birds

## Name long
Relationship between a new open-source river network database in Czechia and the long-term collection of waterbird monitoring data.

## Annotation

This project explores potential relationships between the recently published open-source river network database for Czechia and the long-term waterbird monitoring data collected across the country. **The main idea is to examine whether spatial patterns in river morphology, network connectivity, and habitat structure correspond with long-term patterns in waterbird distribution and abundance.**

The river network database is available at:
https://tokyamokrady.aopk.gov.cz/

Information about the long-term waterbird monitoring program can be found at:
https://www.waterbirdmonitoring.cz/
